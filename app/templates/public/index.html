Welcome to our RISC-V Simulator!!

This project is used to implement a RISC-V simulator to trace the execution of the RV32I Instructions.

  • To use the simulator, please go to the "Test" tab to upload your input file.
  • After pressing Run, the output will appear in the "Output" tab.
  • The output shows the name of the current executing instruction followed by the program counter value, the contents of the register file and the memory contents.
